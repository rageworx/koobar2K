You may need to import these prebuild libraries in your build up system.

1. FLTK 1.3.2 ( build with OpenGL supported )
   a. libfltk.a
   b. libfltk_forms.a
   c. libfltk_images.a
   d. libfltk_jpeg.a (libJPEG)
   e. libfltk_png.a (libPNG)
   f. libfltk_z.a (libZ)

2. mpg123 ( fully build for stl and std option )
  * required name as
     a. libmpg123.a

3. fempg123 ( Front End of mpg123 )
  * required name as
     a. libfempg123.a
